# Sistem Informasi Pemetaan Berbasis Web

Sistem informasi pemetaan berbasis web menggunakan **Leaflet.js** untuk menampilkan lokasi dan informasi geografis secara interaktif.

## Fitur Utama
1. Peta interaktif (Leaflet.js)
2. Marker lokasi dinamis
3. Detail informasi lokasi
4. Sistem admin & user
5. Manajemen data berbasis PHP & MySQL

## Teknologi
1. PHP Native
2. MySQL
3. Leaflet.js
4. JavaScript
5. HTML & CSS

## Instalasi
1. Clone repository
2. Import database (sesuaikan struktur)
3. Rename `config/koneksi.example.php` → `koneksi.php`
4. Sesuaikan konfigurasi database
5. Jalankan di server lokal (XAMPP/Laragon)

## Catatan
Project ini dibuat untuk keperluan pembelajaran dan pengembangan sistem informasi geografis.
